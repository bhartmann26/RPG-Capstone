# RPG-Recommends Personalized Games
## Team Members: John Harris, Benjamin Hartmanm, Patrick Adams
Project Descirption: A video game recommendation system that can be used to find games that match a users taste. Recommends users games that they can then "like" or "dislike" which helps continue to recommend games. Users can also search for and find games that they can add to personal folders. Users can input games they have played manuually or through uploading their steam library.
